## P181 CheckTest
### Q1 色(color)が赤で、インチ(inch)が25の、自転車オブジェクトbicycleを表現するコードを書いてください。
```javascript
const bicycle = {
    color: "赤",
    inch: 26
}
```

### Q2 Q1で書いたオブジェウトから、色(color)の値を取得するコードを書いてください。
```javascript
bicycle.color;
```

### Q3 Q1で書いたオブジェウトのインチ(inch)を16に変更するコードを書いてください。
```javascript
bicycle.inch = 16;
```
